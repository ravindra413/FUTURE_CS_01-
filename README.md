# Web Application Vulnerability Assessment

This project contains a vulnerability assessment conducted on the website http://demo.testfire.net.

## Tools Used
- Nmap
- OWASP ZAP

## Assessment Scope
- SQL Injection
- Security header misconfiguration
- Information disclosure
- Session and cookie security issues

## Outcome
Several vulnerabilities were identified and categorized by risk level. 
Recommendations were provided to strengthen the security of the web application.
